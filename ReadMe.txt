Сервис по парсингу цен на предметы из MMORPG Crossout

Скачивает цены с сайта crossoutdb.com, используя их рест апи, производит расчёты и отображает пользователю
Сервис использует спринг для разрешения зависимостей и монго дб для хранения данных

Немного про классы:

ExtremumService забирает из монги предыдущие цены на вещи и обновляет их при парсинге сайта, так как сам сайт не хранит
цены за всё время

CraftService парсит один большой json, получаемый от сайта и отображает результат после расчётов

SkipSslVerificationHttpRequestFactory - взял с стек оверфлоу, нужен для подключения к сайту без проверки ssl ключей,
в данном случае проверка ssl несущественна

MarketController - тестовый контроллер для проверки доступности приложения

ParserApp - главный класс, поднимает спринг, который в свою очередь запускает сервисы ExtremumService и CraftService
по спринговому планировщику
